# Búlgaree

Este repositório contém os arquivos de instalação do programa Búlgaree.

## Como Instalar

1. Baixe o arquivo `Instalador_Bulgaree.exe` da pasta `dist`.
2. Execute o instalador e siga as instruções na tela.
3. O programa será instalado e um atalho será criado na sua Área de Trabalho.

## Como Desinstalar

Para remover o programa:
1. Vá ao **Painel de Controle** > **Programas** > **Desinstalar um programa**.
2. Encontre **Bulgaree** na lista.
3. Clique em **Desinstalar**.

## Recursos

- **Instalação Limpa**: O instalador configura todos os atalhos necessários.
- **Desinstalação Completa**: O desinstalador remove todos os arquivos e entradas de registro.
